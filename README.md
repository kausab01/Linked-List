# 🔗 LinkedList Examples in Java

## 📌 Project Overview

This project contains multiple examples demonstrating the usage of **LinkedList in Java** using the Java Collections Framework.
Each file in the repository represents a different operation performed on a LinkedList such as insertion, deletion, iteration, searching, reversing, and more.

The purpose of this project is to understand how LinkedList works internally and how it can be used to manage dynamic data efficiently.

---

## 🎯 Objective

The objective of this project is to learn:

* How LinkedList differs from ArrayList
* Dynamic data handling using LinkedList
* Performing CRUD operations on LinkedList
* Traversing and manipulating elements
* Using built-in LinkedList methods effectively
